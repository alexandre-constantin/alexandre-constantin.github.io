Research

My research interests are centered around Signal and image processing techniques with applications to satellite imagery (remote sensing). I have many interests in Machine Learning, Statistics and Time-Frequency techniques.

Projects

Analysis of irregularly sampled signals

The main objective is to be able te remove the re-sampling strategy to analyse raw time-series.

  • Gaussian processes,
  • Classification,
  • Satellite Image Time-Series (SITS).

Understanding size of recurrent network structures

This project aims to study the size of a network with application to time-series and aims to find the minimal size of a network for a given problematic.

  • Recurrent neural networks,
  • Explainable AI,
  • Geoscience.
